  <h2 className="text-3xl font-bold text-gray-900 dark:text-white mb-4 text-center">
    Quick start
  </h2>
  <p className="text-base text-gray-600 dark:text-gray-400 text-center mb-12 max-w-2xl mx-auto">
    Get your first project live in under a minute
  </p>

  <Steps>
    <Step title="Install Bool CLI">
      Install the CLI globally using npm:
      
      <CodeGroup>
      ```bash npm
      npm install -g bool-cli
      ```
      ```bash yarn
      yarn global add bool-cli
      ```
      ```bash pnpm
      pnpm add -g bool-cli
      ```
      </CodeGroup>
    </Step>

    <Step title="Deploy your first project">
      Navigate to any directory with web files and run:
      
      ```bash
      bool shipit
      ```
      
      <Note>
        No API key required! `shipit` creates an anonymous deployment and gives you an instant live URL.
      </Note>
      
      Example output:
      ```bash
      ✔ Shipped "my-app" (12 files)
        https://abc123.bool01.com
        slug: abc123
      ```
    </Step>

    <Step title="Update your project">
      Make changes and run `shipit` again from the same directory:
      
      ```bash
      bool shipit
      ```
      
      Bool CLI automatically updates your existing deployment using the config stored in `.bool/config`.
    </Step>

    <Step title="Set up authentication (optional)">
      For full features like listing projects and custom names, authenticate with your Bool account:
      
      ```bash
      bool auth login
      ```
      
      Get your API key from [bool.com](https://bool.com).
    </Step>
  </Steps>
</div>
